Este CU tiene como objetivo la administración de los Servicios de Banco de SIAT.
Desde el punto de vista de los Bancos que cobran Recursos de la Municipalidad, representan los códigos de Servicios. Con cada uno pueden tener asociado un descuento general.
Desde el punto de vista de SIAT, representan un agrupamiento lógico o funcional de los recursos.

Mediante este CU se permite dar de alta, baja, y modificación de un Servicio Banco.

Scenario Type Detail
Activar/Desactivar Servicio Banco Alternate
Dependiendo del estado del Servicio/Banco, se habilitará una de las dos opciones.
Si está en estado "Activo", se procederá a la desactivación del Servicio/Banco, pasando a estado Inactivo y no pudiendo ser referenciada por un Recurso nuevo.
Si está en estado "Inactivo", se procederá a la activación del Servicio/Banco, pasando a estado Activo, pudiendo ser referenciada por cualquier Recurso nuevo.
Agregar Servicio Banco Alternate
Permite dar de alta un nuevo Servicio Banco, para lo cual se debe ingresar el código y la descripción.

Además:
- Se debe seleccionar la Partida genérica para la cual se distribuiran las transacciones Indeterminadas durante los procesos de Asentamiento. Esta se utilizará cuando no se pueda identificar una partida especifica para el Recurso.

- Se debe seleccionar la Partida de Cuenta Puente. Es la partida de la cual se descargará el total de importe distribuido durante el Asentamiento.

- Se debe indicar si el Servicio Banco estará relacionado a al menos un recurso definido como autoliquidable.

- Se debe indicar el tipo de proceso de asentamiento que le corresponde.
SIAT: Proceso de Asentamiento (Interno)
NO SIAT: Proceso de Asentamiento Delegado (Externo)
MIXTO: Proceso de Asentamiento Interno y Externo. Se asocia a ambos procesos.
Buscar Servicio Banco Alternate
Permite realizar la búsqueda de Servicios Bancos por código y descripción.
Si existen resultados se muestra el Código, Descripción y Estado y las acciones "Ver Servicio Banco", "Modificar Servicio Banco", "Eliminar Servicio Banco", "Activar/Desactivar Servicio Banco".
Para agregar un nuevo Servicio Banco, se presenta la acción "Agregar Servicio Banco".
Eliminar Servicio Banco Alternate
Permite eliminar un Servicio Banco
Modificar Servicio Banco Alternate
Permite modificar todos los dato con que se creo el Servicio Banco.

Permite relacionar el Servicio Banco con los Recursos

Permite asociar Descuentos Generales al Servicio Banco

Este CU tiene como objetivo la administración de las Áreas del Organigrama. Se utiliza para validar quien envía información de novedades de Catastro y para indicar a que área pertenece cada usuario. En este último caso, se vincula con el módulo de solicitudes internas.

Scenario Type Detail
Buscar Área Alternate
Permite consultar las áreas disponibles en el SIAT mediante diferentes criterios de búsqueda. Permite buscar por código y descripción.
Si existen resultados se muestra de cada uno de los registros el código y la descripción del área y las acciones asociadas: "Ver Área" "Modificar Área", "Eliminar Área" y "Activar/Desactivar Área".
Para agregar una nueva área, existe la opción "Agregar Área".
Ver Área Alternate
Permite visualizar información del área seleccionada.
Agregar Área Alternate
Para crear una nueva área, debemos ingresar el código y la descripción.
Eliminar Área Alternate
Elimina en forma física el área seleccionada.
Modificar Área Alternate
Permite actualizar la descripción del área.
Activar/Desactivar Área Alternate
Dependiendo del estado del Área, se habilitará una de las dos opciones.
Si está en estado "Activo", se procederá a la desactivación del Área, pasando a estado Inactivo y no pudiendo ser referenciada nuevamente.
Si está en estado "Inactivo", se procederá a la activación del Área, pasando a estado Activo, pudiendo ser referenciada nuevamente.

Este CU tiene como objetivo la administración de los dominios de atributos.

Representa el Maestro de dominios de atributos que pueden ser utilizados por atributos de objetos imponibles, recursos, contribuyente, etc. Por ej., para el tipo de objeto imponible PARCELON, el atributo zona posee un rango de ZONA1, ZONA2, ZONA3, ZONA4.

El dominio podrá estar definido por extensión, indicando la totalidad de sus valores o podrá estar definido por comprensión indicando una tabla del sistema de donde se tomarán los datos.

Nota: En R1 no se incluye la funcionalidad de linkear un Dominio a una tabla.

Scenario Type Detail
Buscar Dominios Alternate
Permite consultar los Dominios disponibles en el SIAT por código o descripción.
Si existen resultados se muestra de cada uno de los registros el código, la descripción y las acciones asociadas: "Ver Dominio", "Modificar Dominio", "Eliminar Dominio" y "Activar/Desactivar Dominio".
Para agregar un nuevo dominio, existe la opción "Agregar Dominio".
Ver Dominio Alternate
Proporciona información del Dominio seleccionado y la lista de los Dominios Atributos Valor (valores propios del dominio) asociados.
Por cada Dominio Atributo Valor se presentará la acción "Visualizar Dominio Atributo Valor".
Agregar Dominio Alternate
Para crear un nuevo dominio se debe ingresar el código, la descripción, el tipo Atributo y el nombre de la clase que lo manejará.
Modificar Dominio Alternate
Presenta el Dominio para editar y asociar los Dominios Atributos Valor correspondientes
Eliminar Dominio Alternate
Elimina en forma física el Dominio seleccionado si no está siendo referenciado.
Activar/Desactivar Dominio Alternate
Dependiendo del estado del Dominio Atributo, se habilitará una de las dos opciones.
Si está en estado "Activo", se procederá a la desactivación del Dominio Atributo, pasando a estado "Inactivo" y no pudiendo ser referenciado nuevamente.
Si está en estado "Inactivo", se procederá a la activación del Dominio Atributo, pasando a estado "Activo", pudiendo ser referenciado nuevamente.

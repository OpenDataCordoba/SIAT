Este CU tiene como objetivo la administración de Atributos.

Un atributo representa una propiedad de una entidad del sistema. Por ej. atributos de tipo de objeto imponible, de contribuyente, etc.

Scenario Type Detail
Buscar Atributo Alternate
Permite consultar los atributos disponibles en el SIAT mediante diferentes criterios de búsqueda.
Permite buscar por Código y Descripción.
Si existen resultados se muestra de cada uno de los registros el Código, la Descripción, Tipo Atributo y el Estado y las acciones asociadas: "Ver Atributo", "Modificar Atributo" , "Eliminar Atributo" y "Activar/Desactivar Atributo".
Para agregar un nuevo atributo, existe la opción "Agregar Atributo".
Ver Atributo Alternate
Permite visualizar información del atributo seleccionado.
Agregar Atributo Alternate
Para crear un nuevo atributo, debemos ingresar los siguientes datos:
- Código
- Descripcion
- Tipo Atributo
- Dominio asociado al Atributo
- Máscara de visualización

* Cuando se seleccione un Tipo Atributo, el sistema filtrará los Dominios que correspondan con el Tipo seleccionado.
Eliminar Atributo Alternate
Elimina en forma física el Atributo seleccionado siempre y cuando no esté siendo refenciado.
Modificar Atributo Alternate
Permite actualizar todos los datos del Atributo con excepción del Código. La carga es similar al "Agregar Atributo".
Activar/Desactivar Atributo Alternate
Dependiendo del estado del Atributo, se habilitará una de las dos opciones.
Si está en estado "Activo", se procederá a la desactivación del Atributo, pasando a estado Inactivo y no pudiendo ser referenciado nuevamente.
Si está en estado "Inactivo", se procederá a la activación del Atributo, pasando a estado Activo, pudiendo ser referenciado nuevamente.

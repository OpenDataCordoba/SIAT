Este CU permite la administración de Planes de Control Fiscal.
Existira un unico plan de tipo Coyuntural, sin vigencia, el cual contendra los Operativos: Cierre Comercios, Devoluciones/Compensaciones, Transferencia de Fondo de Comercio, Concursos y Quiebras y Coyuntural. En estos operativos se generarán todas las Ordenes de Control Fiscal asociado a los "trámites" antes mencionados.
Nota: Entregado en R3 y ajustado para R4.

Scenario Type Detail
Búsqueda de Planes de Control Fiscal Alternate
Permite consultar los planes de control fiscal disponibles en el SIAT por la descripción, estado, fecha de inicio, fecha fin
Si existen resultados se muestra de cada uno de los registros la descripción del plan, la fecha de inicio y finalización del mismo y el estado, y las acciones asociadas: "Visualización de Plan de Control Fiscal", "Modificación de Plan de Control Fiscal" y "Eliminación de Plan de Control Fiscal".
Para agregar un nuevo plan, existe la opción "Agregar Plan de Control Fiscal".
Agregar Plan de Control Fiscal Alternate
Para crear un nuevo Plan de Control Fiscal se debe ingresar Nro de plan, Objetivo, Fundamentos, Propuesta, Metodología de trabajo, Necesidades, Resultados esperados, Fecha de Creación.. El Plan creado tendrá el estado Abierto.
Eliminación de Plan de Control Fiscal Alternate
Permite la eliminación del plan de control fiscal seleccionado.

Antes de eliminar el Plan se debe validar que no hayan Operativos de Investigación asociados al mismo, en este caso mostrar error y no eliminar
Modificación de Plan de Control Fiscal Alternate
Para actualizar información del Plan de Control Fiscal seleccionado se presenta la opción "Modificar Plan de Control".
Se permite modificar los datos del Plan y cambiar el estado a Cerrado, momento a partir del cual no se permite trabajar con el Plan (crear operativos, emitir ordenes de control, etc).
Visualización de Plan de Control Fiscal Alternate
Permite visualizar información asociada al plan de control fiscal seleccionado.

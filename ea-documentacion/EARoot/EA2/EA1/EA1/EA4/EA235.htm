Scenario Type Detail
Administrar deuda en CyQ Alternate
- El Usuario ingresa Recurso y Cuenta
- El Sistema recupera la deuda existente para la cuenta seleccionada y permite seleccionarla siempre y cuando se encuentre en la vía Judicial y no esté ni reclamada ni indeterminada ni exenta de pago.
- El Usuario selecciona la deuda que desea enviar a CyQ
- El sistema solicita la fecha a la cual se actualizará la deuda y el procedimiento al cual se desea enviar.
- El usuario ingresa lo datos solicitados
- El sistema determina la actualización de la deuda a la fecha indicada y graba la relación con el procedimiento.

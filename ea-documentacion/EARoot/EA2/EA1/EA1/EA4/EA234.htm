Scenario Type Detail
Activar/desactivar procedimiento Alternate
- El usuario selecciona un procedimiento e ingresa a la la opción de Activar/Desactivar:
- El sistema recupera el procedimiento indicado y solicita la confirmación de la Activación/desactivación.
- El usuario confirma
- El sistema procede con la acción indicada.
Alta de Procedimiento Alternate
- El usuario selecciona la opción de realizar alta de procedimiento
- El sistema solicita el ingreso de los siguientes datos: Número, Año, Fecha, Carátula, Número de expediente, año de expediente y observaciones.
Buscar Procedimientos Alternate
- El sistema permite realizar búsquedas de procedimientos existentes por los siguientes criterios: Número, Año y Carátula.
- El Usuario pude ingresar cualquiera de estos parámetros y realizar la búsqueda.
- El Sistema obtiene los procedimientos existentes para los parámetros ingresados. De cada uno muestra: Número, Año, Carátula y Estado.
Sobre cada uno se habilitan las opciones de: Ver, Modifica, Eliminar, Activar/Desactivar.
Adicionalmente se habilita la opción de agregar procedimiento
Eliminar procedimiento Alternate
- El usuario selecciona un procedimiento e ingresa a la la opción de eliminar:
- El sistema recupera el procedimiento indicado y solicita la confirmación de la eliminación.
- El usuario confirma
- El sistema elimina el procedimiento

Validaciones:

- Para poder eliminar un procedimiento, el mismo no debe estar referenciado por ningúna deuda
Modificar procedimiento Alternate
- El usuario selecciona un procedimiento e ingresa a la la opción de modificar:
- El sistema recupera el procedimiento indicado y solicita la modificación de los siguientes datos: Número, Año, Fecha, Carátula, Número de expediente, año de expediente y observaciones.

Este CU tiene por objetivo registrar un reclamo ante la falta de asentamiento de pago de cuota de convenio. En este caso, a diferencia del reclamo en la deuda, no se genera ninguna interacción con varios_web, sino que se registra el reclamos en el ámbito de SIAT.

Scenario Type Detail
Generar Reclamo Asentamiento de Cuota Alternate
Para cada cuota impaga el usuario podrá seleccionar dicha cuota y realizar un reclamo por falta de asentamiento. Esta cuota se coloca en estado "En Análisis" o "Solicitando información" hasta tanto un usuario de Balance la confirme y cambie a "Pago a verificar".
Para generar dicho reclamo se deben ingresar los siguientes datos:
- Fecha de Pago, Importe Pagado, Institución.
- Nombre, Apellido, Tipo y Nro de Documento, Teléfono, Correo Electrónico.
- Observación
Para la gestión de los reclamos se invoca el CU Administrar Reclamos.

Scenario Type Detail
Determinar DReI Alternate
Usuario CMD/CA determina deuda de DReI, para ello selecciona el botón de determinar deuda del estado de Cuenta DReI en el período correspondiente. El sistema chequea si ya hubo una declaración anterior presentada, si no la hubo crea una del tipo DDJJ original y se ingresa:
Cantidad de personal, Cada actividad declarada por el contribuyente.
Para cada actividad se ingresa la base imponible y la alícuota correspondiente.
El sistema obtiene el pago mínimo mediante el parámetro cantidad de personal, compara este valor con el obtenido de realizar la sumatoria de base imponible x alicuota recuperando el valor máximo.
Se ingresa el porcentaje correspondiente para mesas y sillas, y adicional publicidad.
El total a determinar se calcula como el subtotal restandole el valor ingresado en otros pagos (retenciones, saldo de meses anteriores, etc).
El sistema calcula un subtotal igual a la suma del valor máximo, valor máximo x porcentaje mesas y sillas y valor máximo x porcentaje adicional publicidad

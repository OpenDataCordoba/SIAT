Mediante este caso de uso se permite imprimir la constancia de cierre definitivo de una Comercio.
A través de la liquidación de deuda, una vez que el comercio tiene estado "Con Cierre Definitivo" se hará visible un botón con la opción Imprimir Constancia de Cierre, el sistema presentará en pantalla un pdf conteniendo la información de cierre.

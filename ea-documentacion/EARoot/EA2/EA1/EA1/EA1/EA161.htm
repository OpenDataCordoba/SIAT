Este CU permite realizar la búsqueda de todas las Cuentas de un Objeto Imponible y acceder al Estado de la Cuenta y la Liquidación de Deuda correspondiente.
Las Cuentas obtenidas son todas aquellas que están relacionadas con el Objeto Imponible. Por ejemplo, dada una parcela, podrán obtenerse cuentas de: TGI, CdM, Tasa de Revisión de planos.

Scenario Type Detail
Obtener Liquidación de Deuda de Objeto Imponible Alternate
Para obtener la Liquidación de Deuda de un Objeto Imponible se deben ingresar los siguientes parámetros:

- Tipo Objeto Imponible (requerido)
- Clave


Se obtiene los Objetos Imponibles correspondientes con los parámetros ingresados por el usuario.

Cuando el usuario seleccione uno de los registros, se mostrarán las cuentas relacionadas con ese objeto imponible para que el usuario seleccione si desea ver la Liquidación de Deuda por Recurso/Cuenta (incluye el CU "Obtener Liquidación de Deuda por Recurso/Cuenta" o Estado de Cuenta (incluye el CU "Estado de Cuenta").

Este CU permite realizar el Rescate masivo de convenios. Su utilización está prevista para los casos en que se requiera considerar como pagos buenos a todos los pagos a cuenta registrados en un determinado período para un determinado plan.


Rescatar un convenio consiste en volver a colocarlo vigente para permitir la gestión normal del mismo.

Scenario Type Detail
Rescatar Convenio Alternate
El CU permite ingresar el Caso, la Fecha de Vigencia del rescate y la Observación para generar el Rescate del convenio. Una vez ingresados:
- Se registra un Rescate de Convenios (igual a los generados en Administrar Rescate Masivo de Convenios), con la Fecha del día, el Recurso y el Plan de Pago asociados al Convenio, la Observación, el Caso y la Fecha de Vigencia.
- Se genera una nueva Corrida del proceso correspondiente al Rescate de Convenio, en estado "En Preparación", la cual se asocia al Rescate de Convenio antes creado.
- Se genera una nueva Selección Almacenada con el Convenio como único detalle y se la asocia al Rescate de Convenio.
- Se configura la corrida para ejecutarse en forma síncrona.

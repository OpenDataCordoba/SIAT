El objetivo del proceso es generar la estructura de datos necesaria para el input del proceso de generación de declaraciones juradas y trasacciones de pago.
Implica:
- Obtener los envíos que no se encuentren ya procesados leyendo los datos existentes en la base creada para explotar archivos MULATOR (validando la clave compuesta org_recaud_id y envio_id)
- Validar que las transacciones informadas en el envío no se hayan procesado previamente (que no existe el mismo ID_TRANSACCION_AFIP) en SIAT.

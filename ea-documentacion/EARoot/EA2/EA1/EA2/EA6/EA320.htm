Un Uso describe la utilización que se le dará a una parcela, unidad funcional, etc.
Este mantenedor permite establecer una relación entre los usos utilizados en Catastro y los utilizados en Contribución de Mejoras.
El sistema utilizará esta relación cuando se agregan detalles a las planillas de cuadras ejecutadas, tratando de vincular el uso cargado por catastro (e.j. Baulera) con alguno de CdM (ej. Residencial), y asi automatizar la carga de estos.

Scenario Type Detail
Buscar Uso Alternate
Permite consultar los usos disponibles en el SIAT mediante diferentes criterios de búsqueda. Permite buscar por descripción de uso.
Si existen resultados se muestra de cada uno de los registros la descripción, factor, uso de catastro y las acciones asociadas: "Ver Uso", "Modificar Uso", "Eliminar Uso", "Activar/Desactivar Uso".
Para agregar un nuevo Uso, existe la opción "Agregar Uso".
Ver Uso Alternate
Permite visualizar información del Uso seleccionado.
Modificar Uso Alternate
Permite actualizar todos los datos asociados al Uso.
Eliminar Uso Alternate
Elimina en forma física el registro de uso seleccionado en caso de no tener registros relacionados.
Activar/Desactivar Uso Alternate
Dependiendo del estado del Uso se habilitará una de las dos opciones.
Si está en estado "Activo", se procederá a la desactivación del Uso, pasando a estado "Inactivo" y no pudiendo ser referenciado nuevamente.
Si está en estado "Inactivo", se procederá a la activación del Uso, pasando a estado "Activo", pudiendo ser referenciado nuevamente.
Agregar Uso Alternate
Para crear un nuevo Uso.
Se debe:
. ingresar la descripción
. ingresar el factor
. ingresar una lista separada por comas de los usos de catastro

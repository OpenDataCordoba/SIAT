El objetivo de este caso de uso es realizar la baja lógica (desactivación) o alta lógica (activación) de un objeto imponible.

Scenario Type Detail
Activar/Desactivar Alternate
Se recibe el Objeto Imponible y la acción a realizar.
Se activa o desactiva (se cambia el estado) el Objeto Imponible en función de la acción recibida como parámetro.

El Sistema obtiene la lista de los Recursos asociados al tipo de objeto imponible relacionado.
Si posee un recurso principal, se verifica la configuración existente para las cuentas.
Si la acción es "Desactivar", verifica la configuración:
Si "realiza la baja de la cuenta cuando se realiza la baja del objeto imponible asociado"
Obtiene la Cuenta asociada con el recurso principal
Invoca al CU: "Activar/Desactivar Cuenta", con la Cuenta como parámetro y con la acción: Desactivar.

Si la acción es "Activar", verifica la configuración:
Si "realiza el alta de cuenta cuando se realiza el alta del objeto imponible asociado"
Obtiene la Cuenta asociada con el recurso principal
Invoca al CU: "Activar/Desactivar Cuenta", con la cuenta como parámetro y con la acción: Activar

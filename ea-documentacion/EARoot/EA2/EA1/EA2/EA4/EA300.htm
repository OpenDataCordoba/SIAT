Appears In: Cuentas Contributivas
Este CU tiene como objetivo la administración del padrón de cuentas contributivas.
La cuenta contributiva representa la entidad del sistema contra la cual se genera la deuda. Una cuenta siempre corresponde a un único Recurso. Una cuenta puede estar asociada a un Objeto Imponible o no, según sea la característica del Recurso al cual pertenece. Por ejemplo, las cuentas del recurso TGI se asocian a parcelas; las cuentas del recurso: Tasa de Desinfección de Ropa no poseen objeto imponible asociado. Una cuenta posee otras cuentas asociadas del mismo recurso. Por ejemplo: las unificaciones y desgloses de TGI. Una cuenta posee relaciones implícitas con otros Recursos en función de compartir el mismo Objeto Imponible.

Mediante este caso de uso se permiten modificar: Datos generales de la Cuenta, Titulares de la Cuenta y Domicilio de envío. Las acciones de de creación, eliminación y modificación de titulares responden a la configuración del recurso relacionado. Por ejemplo: Al realizar el alta de una cuenta del recurso: Tasa de Revisión de Planos, automáticamente, se trasladarán los titulares de la cuenta de TGI que correspondan al Objeto Imponible (Parcela) relacionado con la Catastral de la cuenta que se desea crear.

Scenario Type Detail
Buscar Cuenta Alternate
Permite consultar las cuentas existentes en el SIAT mediante diferentes criterios de búsqueda. Permite buscar por Recurso, Nro.Cuenta y Titular.

Se permitirá realizar también una Busqueda Avanzada, donde se podrá buscar por los atributos definidos para cada tipo de objeto imponible, por rango o multivalor según corresponda. (siempre y cuando el atributo sea atributo de búsqueda). Los atrbutos de búsqueda dependerán del Tipo de Objeto Imponible asociado al Recurso indicado como parámetro. En caso de no indicarse un Recurso, no se podrá referenciar ningún Tipo de Objeto imponible y por lo tanto, no se podrá realizar la búsqueda avanzada.

Si existen resultados se muestra de cada uno de los registros la Fecha de Alta, la Fecha de Baja (en caso de estar inactiva), el Recurso asociado y el Número de cuenta y las acciones asociadas: "Ver Cuenta", "Modificar Cuenta","Eliminar Cuenta", "Activar/Desactivar Cuenta".


Para agregar manualmente una nueva cuenta, existe la opción "Agregar Cuenta".
Ver Cuenta Alternate
Permite visualizar información de la Cuenta seleccionada y la lista de Titulares de la misma.
Agregar Cuenta Alternate
Para crear manualmente una nueva cuenta se deberá:

- Indicar el Recurso, de una lista de Recursos. El Sistema solo mostrará los Recursos que posean habilitada la opción de: Alta Manual en la configuración del mismo.

- Seleccionar el Objeto Imponible: Se permitirá realizar una búsqueda entre todos los Objetos Imponibles del Tipo de Objeto Imponible asociado al Recurso previamente seleccionado.
- Nro. de Cuenta: Deberá ser ingresado si el Recurso posee el atributo de generación de Nro. de Cuenta "Manual".
- Código de Gestión Personal: Deberá ser ingresado si el Recurso admite generación de Código de Gestión Personal "Manual".
- Fecha Alta

- Domicilio de Envío:
- Localidad
- Calle
- Número
- Letra
- Bis
- Ref. Geográfica
- Piso
- Departamento
- Monoblock
- Teléfono

Para validar el domicilio ingresado, se invoca al CU: Mantenedor Domicilio

Una vez obtenidos todos los datos para realizar el alta de cuenta, se Invoca al CU: "Alta de Cuenta"
Eliminar Cuenta Alternate
Elimina en forma física la Cuenta seleccionada cuando no está siendo referenciada.
Modificar Cuenta Alternate
Presenta la Cuenta para modificar con la siguiente información:
Datos de la cuenta
- Recurso
- Número de Cuenta
- Código de gestión personal
- Fecha Alta
- Fecha Baja
Para modificar los datos generales de la cuenta, se presenta la opción: Modificar Datos Generales

Domicilio de Envío
- Datos del domicilio
Para modificar el Domicilio de envío, se presenta la opción: Modificar Domicilio de Envío

Lista de Titulares
- Lista de Titulares. Para cada uno las acciones "Ver Titular", "Modificar Titular" y "Eliminar Titular".
Para la creación de un nuevo Titular asociado a la Cuenta, se presentará la opción "Agregar Titular".

Modificar Datos Generales:
El Sistema permite modificar la fecha de alta y el código de gestión personal en caso de contarse con la posibilidad.

Modificar Domicilio de Envío:
Invoca al CU: Cambiar Domicilio de Envío

El CU tiene como objetivo la administración de Sellados.
Los sellados corresponden a las diferentes Tasas por actuación administrativa que se asocian a distintas acciones realizdas en SIAT.
Cada sellado cuenta con un código y una descripción. Además posee la siguiente información relacionada:
. Forma de calcular el importe: Puede ser un monto fijo o un porcentaje de algún otro valor.
. Acciones sobre SIAT que implicarían el uso del Sellado: Corresponde a la lista de acciones que cobran sellado. Por cada una, se establece un criterio para su selección que generalmente está dado en función de una cantidad de períodos. Por ejemplo, para el caso de una reconfección de deuda de TGI, la determinación de si se debe aplicar un sellado depende de la cantidad de de períodos vencidos en relación a la fecha de reconfección y los vencimientos de TGI.
. Distribución de partidas: Para cada sellado, se indica la forma en que se distribuye en partidas

Scenario Type Detail
Agregar Sellado Alternate

Permite dar de alta un nuevo Sellado. Los datos a ingresar son código y descripción del sellado.
Buscar Sellados Alternate
Permite realizar la búsqueda de Sellados definidos según distintos criterios de búsqueda a ingresar: Nro. Sellado, Descripción Sellado.
En caso de que la consulta tuvo exito, se muestra una lista de todos los Sellados encontrados, mostrando información descriptiva y las acciones "Ver Sellado", "Modificar Sellado", "Eliminar Sellado".
Para agregar un nuevo Sellado se presenta la acción "Agregar Sellado".
Eliminar Sellado Alternate
Permite eliminar (lógicamente) un Sellado.
Modificar Sellado Alternate

Permite actualizar información de un Sellado.

Indicar el importe del sellado con vigencia.

Asociar acciones por las que se debiese cobrar el importe del sellado, para cada acción asociada se debe indicar:
- canal
- recurso
- clasificación de deuda
- via deuda
- fecha de inicio de vigencia
para los cuales debe aplicarse el cobro del sellado

Una vez definidas las acciones para el sellado, se debe indicar para cada una, en caso de que corresponda él o los criterios que determinan el cobro del sellado para la acción dependiendo de una cantidad considerada como parámetros del criterio.
Ver Sellado Alternate
Proporciona información detallada del Sellado seleccionado.

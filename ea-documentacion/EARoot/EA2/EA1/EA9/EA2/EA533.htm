El objetivo de este CU es la administración de Procuradores.
Un Procurador es el responsable de gestionar el cobro de una deuda pendiente, que fue transferida desde la vía administrativa a la vía judicial.
En SIAT se considera que un procurador puede trabajar con deuda de distintos recurso. De este modo, se debe indicar para el o los recursos que trabaja definiendo en cada caso, los datos para el criterio de asignación correspondiente y el esquema de comisiones pactado.
Para el caso de TGI, el criterio de asignación es por rango de catastrales. De este modo, para cada procurador que trabaje con deuda de TGI, se deberá indicar cual es el rango de catastrales que le corresponde.

Scenario Type Detail
Búsqueda de Procuradores Alternate
Esta operación incluye la consulta de Procuradores según distintos criterios de búsqueda definidos. Algunos de los filtros de búsqueda a ingresar pueden ser: apellido y nombre, tipo y nro. de documento, CUIL/CUIT.
Por cada Procurador encontrado, según los criterios de búsqueda ingresados, se muestran datos del procurador.
En caso de no existir el Procurador buscado, puede ser agregado a través de la opción "Agregar Procurador".
Si se encontró el Procurador buscado, se presentan distintas acciones relacionadas con el mismo: "Visualizar Procurador", "Modificar Procurador", "Eliminar Procurador" y "Activar/Desactivar Procurador"
Visualizar Procurador Alternate

Permite visualizar información del Procurador seleccionado junto con los Recursos asociados con sus respectivas vigencias y criterio de asignaciòn definido.
Por cada Recurso asociado se permite consultar los Porcentajes de Comisiones definidas.
Agregar Procurador Alternate

Permite agregar un nuevo Procurador. Se deberá:
+ seleccionar una Persona. Incluye el CU Mantenedor Persona.
+ ingresar el teléfono y Horario de Atención del Procurador
+ realizar el alta del domicilio: Incluye el CU Mantenedor Domicilio
Eliminar Procurador Alternate

Permite eliminar físicamente un Procurador (sigue existiendo en el sistema como Persona).
Modificar Procurador Alternate

Permite actualizar información del Procurador, y asociarle Recursos.
Para agregar un nuevo Recurso al Procurador, se debe seleccionar el Recurso e ingresar la vigencia.
Por cada Recurso asociado al Procurador se presentan las siguientes acciones:
."Visualizar Recurso": presenta informaciòn del Recurso y la lista de Comisiones asociadas.
."Modificar Recurso": permite modificar la vigencia del Recurso asociado y administrar los Porcentajes de Comisión correspondientes. Para actualizar las comisiones, se debe ingresar:
+ fecha vencimiento deuda desde
+ fecha vencimiento deuda hasta
+ porcentaje de comision
+ vigencia.
Podrà modificar el porcentaje de la comisión mediante la opciòn "Modificar Comisión" y eliminarla mediante "Eliminar Comisiòn".
."Eliminar Recurso": quita la asociaciòn entre el Procurador y Recurso seleccionado.
Activar/Desactivar Procurador Alternate

Permite activar/desactivar un Procurador según su estado. Únicamente podrá desactivar el Procurador si no tiene asociada la gestión de deuda judicial.
